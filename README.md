# 🤖 Anthropic Agent Methodology | AI 智能体开发方法论

Full-stack methodology for building AI agents based on Anthropic's research. Covers architectural patterns, domain-specific skill injection, and production-grade evaluation.
基于 Anthropic 核心研究的全栈 AI Agent 开发方法论。涵盖了架构模式（工作流 vs. 智能体）、领域特定技能注入、多智能体编排以及生产级评估体系。

---

### 🧠 Core Concepts | 核心概念

-   **🏗️ Architectural Patterns** | **架构模式**：深入理解 Prompt Chaining、Routing 和 Orchestrator-Workers 等核心构建块。
-   **🛠️ Skill-Driven Evolution** | **技能驱动演进**：通过“技能库”模式，将领域知识动态注入智能体，实现能力的精准扩展。
-   **📡 ACI Design** | **智能体交互设计**：优化 Agent-Computer Interface，让 AI 能够更精准地操控环境与工具。
-   **📈 Production Evals** | **生产级评估**：建立基于 LLM-as-a-Judge 的评估流水线，确保系统的稳定性与可靠性。
